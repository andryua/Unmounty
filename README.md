# Unmounty 🔌💾

**Unmounty** — це зручна утиліта для macOS, яка дозволяє швидко та безпечно відмонтовувати та вилучати зовнішні диски, карти пам'яті (SD cards), мережеві диски та образи дисків (DMG) безпосередньо з рядка меню (Menu Bar) або Dock.

---

## ✨ Особливості (Features)

*   **⚡️ Швидке вилучення**: Вилучайте окремі диски або всі підключені накопичувачі в один клік.
*   **📐 Адаптивний інтерфейс**: Розмір вікна автоматично підлаштовується під кількість змонтованих дисків. Якщо накопичувачів немає — вікно залишається компактним.
*   **🛌 Безпечний сон (Auto-Eject before sleep)**: Додаток автоматично розмонтовує зовнішні диски перед тим, як ваш Mac переходить у режим сну, запобігаючи помилкам файлової системи.
*   **⚙️ Гнучкі налаштування**:
    *   Запуск при старті системи (Launch at login).
    *   Режими відображення: тільки в рядку меню (Menu bar), тільки в Dock, або в обох місцях одночасно.
    *   Автоматичне приховування іконки в Menu bar, якщо немає підключених дисків.
*   **🌐 Локалізація**: Повна підтримка української та англійської мов.
